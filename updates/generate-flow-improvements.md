# 画像生成フロー改善

## やりたいフロー
1. テンプレートで画像生成
2. 生成した画像を分析（JSON化）
3. JSON編集で細かく修正

## 必要な変更

### 1. 生成結果から編集モードへの連携
- 生成結果に「この画像を編集する」ボタンを追加
- クリック時: 生成画像をeditModeのアップロード済み状態にセット → 分析可能に

### 2. プロンプト確認・承認フロー
- テンプレート入力後「プロンプトを確認」ステップを表示
- プロンプトの各パートの解説表示
- プロンプトをテキストエリアで直接編集可能
- 「承認して生成」ボタンで実行

### 3. 分析が出なくなった問題
- editModeContentで囲んだことによる影響を確認・修正

## 完了後
実装完了したらこのファイルを `updates/done/` に移動すること
